# Pixel Tayama

An unofficial fan-made pixel Codex pet inspired by Tayama from *Smoking Behind the Supermarket
with You*. Her animation stages a quiet nighttime smoke break against a compact, generic
supermarket rear-service vignette and follows the native Codex desktop v2 contract.

![Idle smoking animation](preview/00-idle.gif)

## Install

From the repository root:

```bash
openpets install tayama
```

Open Codex **Settings -> Pets**, choose **Refresh**, select **Pixel Tayama**, and choose
**Wake Pet**.
