# Pixel Makise Kurisu

An unofficial fan-made pixel Codex pet inspired by Makise Kurisu from *Steins;Gate*. The animation
uses the native Codex desktop v2 contract: nine work-state rows plus a 16-direction look loop.

![Idle animation](preview/00-idle.gif)

## Install

From the repository root:

```bash
openpets install makise-kurisu
```

Open Codex **Settings → Pets**, choose **Refresh**, select **Pixel Makise Kurisu**, and choose
**Wake Pet**.
