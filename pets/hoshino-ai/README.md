An unofficial fan-made pixel Codex pet inspired by жҳҹйҮҺзҲұ (Ai Hoshino). The animation uses the
native Codex desktop v2 contract: nine work-state rows plus a 16-direction look loop.

![Idle animation](preview/00-idle.gif)

## Install

From the repository root:

```bash
openpets install hoshino-ai
```

Open Codex **Settings вҶ’ Pets**, choose **Refresh**, select **Pixel жҳҹйҮҺзҲұ**, and choose **Wake Pet**.
