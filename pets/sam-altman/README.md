# Kneeling Sam Altman

An unofficial, respectful pixel caricature of Sam Altman. The animation uses the native Codex
desktop v2 contract: nine work-state rows plus a 16-direction look loop. Its resting identity is a
calm one-knee-down pose; travel states use a compact crouched shuffle so movement remains legible.

![Idle animation](preview/00-idle.gif)

## Install

From the repository root:

```bash
openpets install sam-altman
```

Open Codex **Settings → Pets**, choose **Refresh**, select **Kneeling Sam Altman**, and choose
**Wake Pet**.
